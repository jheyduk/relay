       ```kotlin
       package dev.heyduk.relay.util

       /**
        * FZF-style fuzzy matching. Returns a score > 0 if all characters in [query]
        * appear in [target] in order (case-insensitive). Higher scores for:
        * - Consecutive character matches
        * - Start-of-word matches (after '/', '-', '_', '.')
        * Returns 0 if no match.
        */
       fun fuzzyMatch(query: String, target: String): Int
       ```
       Implementation: iterate query chars against target chars. Score +1 per match, +2 bonus for consecutive matches, +3 bonus for start-of-word match (char at index 0 or preceded by `/`, `-`, `_`, `.`). Return 0 if not all query chars found.

    3. Create `androidApp/src/main/java/dev/heyduk/relay/presentation/session/CreateSessionViewModel.kt`:
       ```kotlin
       package dev.heyduk.relay.presentation.session

       class CreateSessionViewModel(
           private val relayRepository: RelayRepository
       ) : ViewModel() {

           data class CreateSessionUiState(
               val phase: DialogPhase = DialogPhase.LOADING,
               val directories: List<DirectoryEntry> = emptyList(),
               val filteredDirectories: List<DirectoryEntry> = emptyList(),
               val searchQuery: String = "",
               val selectedPath: String = "",
               val kuerzel: String = "",
               val kuerzelError: String? = null,
               val flags: String = "",
               val flagsEnabled: Boolean = true,
               val isCreating: Boolean = false,
               val errorMessage: String? = null,
               val createdKuerzel: String? = null  // non-null = success, navigate
           )

           enum class DialogPhase { LOADING, DIRECTORY_SELECTION, CUSTOM_PATH, CONFIRMATION }
       ```

       ViewModel responsibilities:
       - `loadDirectories()`: called on dialog open, sends `relayRepository.sendListDirectories()`, collects `relayRepository.updates` filtering for `RelayMessageType.DIRECTORY_LIST`, populates directories and defaultFlags, transitions to DIRECTORY_SELECTION phase
       - `onSearchQueryChanged(query: String)`: updates searchQuery, runs `fuzzyMatch()` on all directories against query, sorts by score descending, updates filteredDirectories. Empty query shows all directories.
       - `onDirectorySelected(entry: DirectoryEntry)`: sets selectedPath=entry.path, kuerzel=entry.name (basename), transitions to CONFIRMATION phase
       - `onCustomPathSelected()`: transitions to CUSTOM_PATH phase
       - `onCustomPathConfirmed(path: String)`: sets selectedPath=path, derives kuerzel from last path segment, transitions to CONFIRMATION
       - `onKuerzelChanged(value: String)`: updates kuerzel field, clears kuerzelError
       - `onFlagsToggled()`: toggles flagsEnabled, updates flags (either defaultFlags or "")
       - `onCreateSession()`: validates kuerzel is non-empty (sets kuerzelError="Kuerzel is required" if empty), sends `relayRepository.sendCreateSession(path, kuerzel, flags)`, sets isCreating=true, collects updates for `RelayMessageType.SESSION_CREATED`, on success sets createdKuerzel, on error sets errorMessage
       - `onBackToDirectorySelection()`: resets to DIRECTORY_SELECTION phase
       - `onDismissError()`: clears errorMessage

       Use `viewModelScope.launch` for coroutines. Collect updates with a timeout (10 seconds) for server responses — show error on timeout.

       **DIRECTORY_SELECTION phase:**
       - `OutlinedTextField` at top for search query with leading search icon, placeholder "Search directories..."
       - `LazyColumn` showing `uiState.filteredDirectories` — each item as a `ListItem` with:
         - headlineContent: `entry.name` (bold)
         - supportingContent: `entry.path` (secondary color, smaller)
         - onClick: `viewModel.onDirectorySelected(entry)`
       - Fixed item at bottom of list: `ListItem` with text "Custom Path..." and leading icon `Icons.Default.Edit`, onClick: `viewModel.onCustomPathSelected()`
       - If filteredDirectories is empty and searchQuery is not empty: show "No matching directories" text centered

       **CUSTOM_PATH phase:**
       - `OutlinedTextField` for path input, label "Project path", placeholder "/Users/..."
       - "Back" text button and "Continue" filled button
       - On Continue: `viewModel.onCustomPathConfirmed(pathInput)`

       **CONFIRMATION phase:**
       - Read-only path display: `Text(uiState.selectedPath)` with label above
       - `OutlinedTextField` for kuerzel, label "Session name (kuerzel)", value `uiState.kuerzel`, `onValueChange = viewModel::onKuerzelChanged`
       - If `uiState.kuerzelError != null`: show error text below kuerzel field in `MaterialTheme.colorScheme.error` color
       - `Row` with `Switch(checked = uiState.flagsEnabled, onCheckedChange = { viewModel.onFlagsToggled() })` and label "--dangerously-skip-permissions"
