   - Content layout: Column with Text(content) + Text(timestamp) in `bodySmall` / `onSurfaceVariant`

     - SnackbarHost for errors
   - Content: `LazyColumn(reverseLayout = true)` per CONTEXT.md decision

     - Each item is a `MessageBubble`

     - verticalArrangement = Arrangement.spacedBy(4.dp)
