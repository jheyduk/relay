1. **PollingService.kt** -- Inject `NotificationHelper` via Koin (`by inject()`). In the `poller.updates.collect` block, AFTER the DB insert, add notification triggering:
   ```kotlin
   when (update.type) {
       RelayMessageType.PERMISSION -> notificationHelper.showPermissionNotification(update)
       RelayMessageType.COMPLETION -> notificationHelper.showCompletionNotification(update)
       else -> { /* no notification for other types */ }
   }
   ```
   This ensures notifications fire for permission and completion messages even when the app is backgrounded (foreground service keeps running).

2. **MainActivity.kt** -- Handle deep-link navigation from notification taps:
   - In `onCreate`, after `setContent`, check `intent?.data` for `relay://chat/{kuerzel}` scheme
   - If present, extract kuerzel and navigate: `navController.navigate("chat/$kuerzel")`
   - Also override `onNewIntent` to handle taps when activity is already running (singleTop scenario):
     ```kotlin
     override fun onNewIntent(intent: Intent) {
         super.onNewIntent(intent)
         // Parse relay://chat/{kuerzel} and navigate
     }
     ```
   - Pass `navController` to a helper or use a LaunchedEffect that observes a state for the deep-link kuerzel.
   - IMPORTANT: Use `LaunchedEffect` with the intent to navigate after the NavHost is composed. Do NOT navigate in `onCreate` before NavHost is ready.

3. **AndroidManifest.xml** -- Add `android:launchMode="singleTop"` to MainActivity so notification taps reuse the existing activity instead of creating a new one. Add deep-link intent filter:
   ```xml
   <intent-filter>
       <action android:name="android.intent.action.VIEW" />
       <category android:name="android.intent.category.DEFAULT" />
       <data android:scheme="relay" android:host="chat" />
   </intent-filter>
   ```

4. **RelayNavGraph.kt** -- Add `deepLinks` parameter to the `composable("chat/{kuerzel}")` route:
   ```kotlin
   composable(
       "chat/{kuerzel}",
       deepLinks = listOf(navDeepLink { uriPattern = "relay://chat/{kuerzel}" })
   ) { ... }
   ```
   This enables Navigation Compose to handle the deep-link URI automatically.
