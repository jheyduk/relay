<objective>
Rewire the app to send audio to the server for transcription and remove all on-device Whisper code.

Purpose: The app no longer needs to bundle the 141 MB Whisper model or compile native C++ code. Recording stays on-device (AudioRecorder.kt), but the WAV file is sent to the Mac server over WebSocket binary frames. The server transcribes and sends back the text. This makes the APK dramatically smaller and transcription faster (Mac Metal GPU vs phone CPU).

Output: Updated WebSocket client with binary send, updated ChatViewModel, deleted WhisperManager/JNI/native code, cleaned build.gradle.kts.
</objective>
