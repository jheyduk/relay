Modify server/relay-server.cjs to add audio receive and transcription:

1. **Config extension** — In loadOrCreateConfig(), add a `whisper_model` field defaulting to `path.join(os.homedir(), '.cache', 'whisper', 'ggml-base.bin')`. Also add `whisper_cli` defaulting to `/opt/homebrew/bin/whisper-cli`. Write both to server.json if not present.

2. **Binary frame detection** — In the `ws.on('message', ...)` handler, detect binary frames. The ws library passes Buffer for binary frames. Check `Buffer.isBuffer(data)` before attempting JSON.parse. Binary frames use this protocol:
   - Bytes 0-1: uint16 big-endian = kuerzel string length
   - Bytes 2..(2+len-1): UTF-8 encoded kuerzel string
   - Bytes (2+len)..end: WAV audio data

   Extract kuerzel and audio buffer from the binary frame.

3. **transcribeAudio function** — Create an async function `transcribeAudio(kuerzel, audioBuffer)`:
   - Write audioBuffer to a temp file: `/tmp/relay-audio-${Date.now()}.wav`
   - Spawn `config.whisper_cli` with args: `['-m', config.whisper_model, '--no-timestamps', '-nt', tmpFile]`
   - Capture stdout, trim whitespace
   - Delete temp file in finally block (fs.unlinkSync)
   - If transcript is non-empty, send JSON to appSocket: `{type: "transcript", session: kuerzel, text: transcript}`
   - If transcription fails, send: `{type: "transcript", session: kuerzel, text: "", error: "reason"}`
   - Log to stderr: `[relay-server] Transcribed N bytes audio for @kuerzel -> "text..."`

4. **Wire binary handler** — In the message handler, when Buffer.isBuffer(data) is true, parse the binary protocol and call transcribeAudio(). When data is a string or parseable JSON, use existing handler logic.

5. **Error handling** — If whisper-cli is not found or model file doesn't exist, log a warning at startup (not fatal — server still works for text). On transcription failure, send error transcript response so the app can show feedback.

IMPORTANT: Do NOT break existing JSON message handling. The binary check must come FIRST since binary frames are not valid JSON.
